### Installation

```sh
$ git clone https://github.com/Scan85/ESDoc.git

$ npm install
```

### Create documentation
* Change to \ESDoc directory
```sh
$ ./node_modules/.bin/esdoc -c esdoc.json
```

### Open documentation
* Change to \ESDoc\docs directory
* open index.html
